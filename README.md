### Question 1: What is Data?

Data is raw, unprocessed facts, figures, symbols that is meaningful to the user.


### Question 2: What is the difference between structured Data and unstructured Data?

Structured data is highly organized, quantitative information formatted in tables with rows and columns (e.g., SQL databases), ideal for easy searching and analysis. Conversely, unstructured data lacks a pre-defined model, comprising text, images, and audio files stored in data lakes or NoSQL systems, requiring advanced tools like AI to analyze.

examples of:
- structured data: financial records and numeric data
- unstructured data: multimedia and text documents

### Question 3: what is schema 

A schema is the word psychologists use to describe a mental framework in which you would file all your knowledge about certain objects, situations, groups of people and even yourself. It would include the whole package of your thinking when you think about something.
